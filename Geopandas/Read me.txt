Navigate to this folder using the anaconda command prompt.

!pip install the files in this folder in order to install geopandas.
e.g:  pip install GDAL-3.2.1-cp38-cp38-win_amd64.whl

These files are compatible for windows 10 64bits and python 3.8.5

The files most be installed in the following order:

	1.- GDAL
	2.- Pyproj
	3.- Fiona
	4.- shapely
	5.- geopandas
